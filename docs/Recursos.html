Recursos para aprender y trabajar con R

Herramientas, guías y materiales seleccionados para acompañar el trabajo práctico del curso: instalación, programación en R, procesamiento de datos, visualización, estadística descriptiva y elaboración de reportes reproducibles.

Preparar entorno

Instalar R, RStudio y Quarto, o usar Posit Cloud como respaldo.

2

Aprender lo básico

Objetos, scripts, paquetes, funciones y estructura de una base.

3

Procesar datos

Importar, limpiar, seleccionar, filtrar, recodificar y resumir datos.

4

Comunicar resultados

Construir tablas, gráficos y reportes reproducibles con Quarto.
