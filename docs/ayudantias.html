Parte II: Ejercicio para preparar la Entrega Parcial

1. Título del ejercicio

Mini investigación en R: Condiciones de trabajo/estudio, uso de tecnología y bienestar cotidiano en personas adultas

La idea es que formulen una pregunta de investigación, exploren una base, limpien datos, recodifiquen variables, creen una variable nueva, calculen descriptivos, hagan gráficos e interpreten resultados. El foco está en construir un análisis descriptivo ordenado, reproducible y bien interpretado.

2. Enunciado

Note

Un equipo de investigación quiere estudiar cómo distintas condiciones cotidianas se relacionan con el bienestar de personas adultas que trabajan, estudian o combinan ambas actividades.

Para eso, se aplicó una encuesta ficticia sobre modalidad de trabajo o estudio, uso de pantallas, horas de sueño, tiempo de traslado, estrés, apoyo social, concentración, uso de inteligencia artificial y bienestar cotidiano.

Tu tarea es construir una mini investigación descriptiva en R. El resultado final debe ser un script ordenado que muestre qué pregunta quieres responder, qué decisiones de limpieza tomaste, qué resultados descriptivos obtuviste y qué puedes concluir sin caer en afirmaciones causales.

3. Problema general de investigación

En distintos ámbitos de la vida adulta, las personas organizan su tiempo entre trabajo, estudio, responsabilidades personales, traslados y uso de tecnología.

Estas condiciones pueden relacionarse con su bienestar cotidiano, descanso, estrés y capacidad de concentración.

En este ejercicio interesa describir cómo se distribuyen estas condiciones según la modalidad principal en que las personas trabajan o estudian: presencial, híbrida o remota.

4. Pregunta de investigación sugerida

Una pregunta posible es:

¿Cómo varían las horas de sueño, el estrés, la concentración percibida y el bienestar cotidiano según la modalidad de trabajo o estudio de personas adultas?

Otra versión posible es:

¿Las personas con mayor carga digital y de tiempo presentan menor bienestar cotidiano que quienes tienen una carga moderada o baja?

5. Hipótesis descriptiva sugerida

Una hipótesis descriptiva posible es:

Las personas con alta carga digital y de tiempo presentan menores horas de sueño, mayor estrés y menor bienestar cotidiano en comparación con quienes tienen una carga moderada o baja.

Warning

Esta hipótesis es descriptiva. No permite afirmar que la carga digital cause menor bienestar. Solo permite observar cómo se distribuyen estas variables en la muestra.

7. Tareas del ejercicio

A continuación se presentan las tareas que deben realizar para construir su mini investigación descriptiva en R.

Tarea 1. Formular el problema de investigación

Ejemplo:

# Tema:
# Modalidad de trabajo/estudio, tecnología y bienestar cotidiano.

# Pregunta:
# ¿Cómo varían el bienestar, las horas de sueño y el estrés
# según la modalidad de trabajo o estudio?

# Hipótesis descriptiva:
# Las personas con mayor carga digital y de tiempo presentan
# menor bienestar cotidiano y mayor estrés percibido.

Deben usar las siguientes funciones:

# ¿Qué variables son numéricas?
# ¿Qué valores extraños o perdidos aparecen?

Pista parcial:

datos_proc <- datos_proc %>%
  mutate(
    edad = case_when(
      edad == 999 ~ NA_real_,
      TRUE ~ as.numeric(edad)
    ),
    
    horas_pantalla_dia = case_when(
      horas_pantalla_dia == -99 ~ NA_real_,
      horas_pantalla_dia > 14 ~ NA_real_,
      TRUE ~ as.numeric(horas_pantalla_dia)
    ),
    
    horas_sueno = case_when(
      horas_sueno == -99 ~ NA_real_,
      horas_sueno > 12 ~ NA_real_,
      TRUE ~ as.numeric(horas_sueno)
    )
  )
Tip

Deben completar el resto de las variables usando la misma lógica.

Tarea 5. Recodificar variables

Deben crear nuevas variables categóricas más fáciles de interpretar.

Variable original Nueva variable Categorías
actividad_principal actividad_cat Trabaja, Estudia, Trabaja y estudia
modalidad modalidad_cat Presencial, Híbrida, Remota
estres estres_cat Muy bajo, Bajo, Medio, Alto, Muy alto
apoyo_social apoyo_cat Muy bajo, Bajo, Medio, Alto, Muy alto
concentracion concentracion_cat Muy baja, Baja, Media, Alta, Muy alta
uso_ia uso_ia_cat Nunca, Pocas veces, Frecuentemente, Siempre

Pista:

datos_proc <- datos_proc %>%
  mutate(
    modalidad_cat = case_when(
      modalidad == 1 ~ "Presencial",
      modalidad == 2 ~ "Híbrida",
      modalidad == 3 ~ "Remota",
      TRUE ~ NA_character_
    )
  )

Tarea 6. Crear variables nuevas

Variable 1: alta carga digital y de tiempo

Crear una variable llamada alta_carga_digital_tiempo.

Debe identificar a personas que cumplen al menos una de estas condiciones:

  • más de 7 horas diarias de pantalla;
  • más de 90 minutos diarios de traslado;
  • más de 50 horas semanales de responsabilidades.

Las categorías serán:

  • "Alta carga"
  • "Carga moderada o baja"

Pista:

datos_proc <- datos_proc %>%
  mutate(
    alta_carga_digital_tiempo = case_when(
      horas_pantalla_dia > 7 |
        traslado_min > 90 |
        horas_responsabilidades > 50 ~ "Alta carga",
      TRUE ~ "Carga moderada o baja"
    )
  )
Variable 2: bienestar agrupado

Crear una variable llamada bienestar_cat.

Condición Categoría
bienestar < 5 Bienestar bajo
bienestar >= 5 & bienestar < 7 Bienestar medio
bienestar >= 7 Bienestar alto

Pista:

datos_proc <- datos_proc %>%
  mutate(
    bienestar_cat = case_when(
      bienestar < 5 ~ "Bienestar bajo",
      bienestar >= 5 & bienestar < 7 ~ "Bienestar medio",
      bienestar >= 7 ~ "Bienestar alto",
      TRUE ~ NA_character_
    )
  )

Tarea 7. Calcular frecuencias y porcentajes

Deben calcular frecuencias para las siguientes variables:

  • actividad_cat
  • modalidad_cat
  • estres_cat
  • uso_ia_cat
  • alta_carga_digital_tiempo
  • bienestar_cat

Ejemplos:

table(datos_proc$modalidad_cat)

prop.table(table(datos_proc$modalidad_cat)) * 100

datos_proc %>%

Tarea 8. Calcular descriptivos numéricos

Deben calcular media, mediana, desviación estándar, varianza, rango y cuartiles para:

  • edad
  • horas_pantalla_dia
  • horas_sueno
  • traslado_min
  • horas_responsabilidades
  • bienestar

Pista:

mean(datos_proc$bienestar, na.rm = TRUE)

median(datos_proc$bienestar, na.rm = TRUE)

sd(datos_proc$bienestar, na.rm = TRUE)

var(datos_proc$bienestar, na.rm = TRUE)

range(datos_proc$bienestar, na.rm = TRUE)

quantile(
  datos_proc$bienestar,
  probs = c(.25, .50, .75),
  na.rm = TRUE
)

summary(datos_proc$bienestar)

psych::describe(datos_proc$bienestar)

Tarea 9. Comparar grupos

Construir una tabla que compare los resultados por modalidad_cat.

La tabla debe incluir:

  • número de casos;
  • media de horas de pantalla;
  • media de horas de sueño;
  • media de traslado;
  • media de estrés;
  • media de concentración;
  • media de bienestar.

Pista:

datos_proc %>%
  group_by(modalidad_cat) %>%
  summarise(
    n = n(),
    media_pantalla = mean(horas_pantalla_dia, na.rm = TRUE),
    media_sueno = mean(horas_sueno, na.rm = TRUE),
    media_traslado = mean(traslado_min, na.rm = TRUE),
    media_estres = mean(estres, na.rm = TRUE),
    media_concentracion = mean(concentracion, na.rm = TRUE),
    media_bienestar = mean(bienestar, na.rm = TRUE)
  )

Luego deben interpretar:

# ¿Qué modalidad presenta mayor promedio de horas de pantalla?
# ¿Qué modalidad presenta mayor tiempo de traslado?
# ¿Qué modalidad presenta mayor bienestar promedio?
# ¿Hay diferencias descriptivas en estrés o concentración?

Tarea 10. Hacer gráficos

Deben construir al menos tres gráficos.

Gráfico 1: barras

Distribución de modalidad.

ggplot(datos_proc, aes(x = modalidad_cat)) +
  geom_bar() +
  labs(
    title = "Distribución según modalidad de trabajo o estudio",
    x = "Modalidad",
    y = "Frecuencia"
  ) +
  theme_minimal()
Gráfico 2: histograma

Distribución del bienestar.

ggplot(datos_proc, aes(x = bienestar)) +

  labs(
    title = "Distribución del bienestar cotidiano",

Bienestar según modalidad.

ggplot(datos_proc, aes(x = modalidad_cat, y = bienestar)) +
